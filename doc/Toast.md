#### Toast 组件 (目前 success fail info loading 模式)

API
| 参数 | 说明 | 类型 | 默认值 |
| ------------ | ---------------------- | ----------------------- | ------ |
| content | 提示内容 | React.Element or String | 无 |
| contentStyle | 提示内容区域样式 | object | 无 |
| duration | 自动关闭的延时，单位秒 | number | 1.5 |
| onClose | 关闭后回调函数 | function | 无 |
| mask | 是否显示透明蒙层 | Boolean | true |
